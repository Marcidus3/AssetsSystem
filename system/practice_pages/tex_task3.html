Добавим шляпу

Работа с отдельными частями модели

При создании ассетов со сложной геометрией могут возникнуть проблемы с редактированием, созданием UV развертки и отрисовкой текстур в трудно доступных частях модели. Для этого ассет разделяют на составные части и работают с каждой из них по-отдельности. Рассмотрим работу с ними в 3D Coat.
Загрузим в новый проект 3D Coat две модели из папки scenes/bake_texture/TeddyBear_LP_UV.obj и Hat_LP.obj. Модель шляпы была создана отдельно от мишки, значит нам необходимо отредактировать ее размер и расположение перед началом работы.

Рис. 1 - Импорт модели

Переходим из режима Paint в Tweak и на левой панели инструментов выбираем Select/Transform. Нажимаем кнопку Select Object в открывшемся окне Tool Options и кликаем на шляпу.

Рис. 2 - Инструмент перемещения

Теперь, когда модель выбрана, нажимаем кнопку Scale и нажимаем на белый куб в центре, чтобы изменить размер.

Рис. 3 - Уменьшение шляпы

Нажимаем на кнопку Move и, используя появившиеся элементы управления, указывающие вдоль каких осей будет происходить передвижение, перемещаем шляпу в область головы мишки.

Рис. 4 - Перемещение шляпы

Для поворота модели нажимаем Rotate.

Рис. 5 - Поворот шляпы

После окончания редактирования моделей переходим в режим UV для создания развертки шляпы. На верхней панели инструментов в выпадающем списке изменяем одинаковые названия разверток default на bear и hat, чтобы переключаться между ними. Также в правой части экрана, во вкладке Paint Objects скрываем модель медведя, чтобы не мешать отображению и работе со шляпой.

Рис. 6 - Изменение названия развертки

Создаем UV развертку шляпы, используя методы, рассмотренные в задании «Создание UV развертки в 3D Coat».

Рис. 7 - Развертка шляпы

Теперь необходимо соединить островки разверток обоих моделей в одну. Снова открываем список и нажимаем Add New UV Set для создания новой развертки, в раскрывшемся окне задаем имя и разрешение текстур.

Рис. 8 - Создание лбщей развертки

Теперь нажимаем на любой из островков UV моделей, а затем на верхней панели инструментов раскрываем список Move Selected и выбираем All. Таким образом островок переместится в новую развертку All. Повторяем данную процедуру для всех островков мишки и шляпы. На развертке all редактируем их расположение и размер.

Рис. 9 - Выбираем новую разертку

Далее загрузим в проект высокополигональные модели File>>Import>> Sculpt/Vertex Painting/Reference. Возьмем файлы scenes/bake_texture/TeddyBear_HP.obj и Hat_HP.obj.
Перейдем в режим Sculpt. Здесь выберем объект Hat_HP во вкладке SculptTree и инструмент Transform на панели слева. Используем элемент управления Gizmo для перемещения и уменьшения шляпы в соответствии с низкополигональной моделью.

Рис. 10 - Перемещение высокополигональной шляпы

Переходим в режим Retopo для выполнения запекания карт нормалей. Выполняем настройку размеров сетки, выполняем запекание и переходим в режим Paint.

Рис. 11 - Запекание карт

При запекании карт AO и Light всегда ставим галочку у Separate paint objects, чтобы процесс выполнялся для каждого из объектов по-отдельности.

Рис. 12 - Включаем запекание по-отдельности

Во вкладке Paint Objects находится список объектов для текстурирования. При нажатии их можно переименовать, иконка глаза отображает видимость объектов, иконка замка предотвращает взаимодействие, а крестик удаляет из сцены.
