Animation Controller

Вместе с геометрией персонажа, были имортированы ее риг и анимация покоя. Чтобы использовать из в рамках игры, необходимо создать объект контроллера анимации, который будет определять, когда следует воспроизводить анимацию.
В папке Animation создаем объект Animation Controller.

Рис. 1 - Создание Animation Controller

Перетаскиваем новый контроллер на модель в рабочей области. Теперь при нажатии на нее, на панели Inspector отображается новая вкладка настройки анимации Animator. В поле Controller отображается созданный нами контроллер.

Рис. 2 - Добавляем контроллер анимации к модели

Кликнем по иконке контроллера на панели Project, чтобы открыть окно работы с анимацией. Здесь уже имеются три узла Entry, Any State и Exit.
Нажемем на импорт модели и в списке ее элементов выберем Idle – анимацию покоя и перетащим ее в окно Animation Controller. Она автоматически соединиться с узлом Entry, определяющим начало какого-либо состояния. Соединим анимацию Idle с узло Any State, нажав на него правой кнопкой мыши и выбрав функцию Make Transition. Кликнем на Idle, чтобы между узлами появилась стрелочка.

Рис. 3 - Настройка Animation Controller

Для проверки работы анимации перейдем в окно игры Game и нажмем на кнопку запуска. Анимация должна проигрываться в зацикленном режиме.
