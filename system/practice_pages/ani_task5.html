Управление анимацией с помощью манипуляторов

Манипулятор Slider

Продолжаем работать над мышью из прошлого задания или открываем проект scenes/anim/Mouse_Slider_Start.max. Создадим манипуляторы в виде слайдеров, чтобы управлять поворотом головы, хвоста и веком. Во вкладке Create выбираем тип объектов Helpers >> Manipulators. Создаем манипулятор Slider и располагаем его в углу рабочего пространства. В настройках указываем название. При перемещении указателя на слайдере происходит изменение его числового значения, которое в свою очередь может быть использовано для управления параметров других объектов.

Рис. 1 - Создание объекта Slider

Создаем три слайдера: для головы, хвоста и глаз.

Рис. 2 - Слайдеры созданы

Теперь соединим данные манипуляторы с объектами с помощью Wire Parameters. Выделяем первый из них, открываем окно Parameter Wiring и нажимаем кнопку Refresh Tree, чтобы выделенный объект стал основным в окне. Основным параметром управления будет значение Value.

Рис. 3 - Обновляем Wire Parameters

В правой части выбираем объект neck и его параметр X Rotation. Устанавливаем связь Connect.

Рис. 4 - Выбираем параметры

Приступаем к корректировке формулы зависимости. Во-первых, уменьшаем степень воздействия value на поворот, умножая его на 0.01.

Рис. 5 - Формула Wire Parameters

Далее открываем настройки манипулятора и включаем режим Select and Manipulate, чтобы взаимодействовать с ним. Теперь перемещаем стрелочку манипулятора и смотрим, при достижении каких значений голова повернется до предела. Запоминаем данные значения и вводим их в параметры Minimum и Maximum слайдера, чтобы ограничить его диапазон.

Рис. 6 - Настройка слайдера

Следующий слайдер для управления глаза соединяем с объектом eyelid, а именно его параметром Z Position. В формуле ставим минус, так как веко должно двигаться сначала вниз.

Рис. 7 - Настройка параметров

В настройках манипулятора оставляем минимальное значение равным нулю, так как веко не будет двигаться выше положения по умолчанию (т.е. когда глаз открыт)

Рис. 8 - Настройка слайдера

Аналогично настраиваем слайдер, управляющий хвостом.

Рис. 8 - Настройка параметров и слайдера

Прежде чем создать тестовую анимацию для слайдеров, нажмем кнопку Key Filters и в открывшемся окне Set Key Filters поставим галочку All, чтобы ключи создавались для всех возможных параметров объектов.
