Топология прочих частей ассета

При создании топологии для сложной геометрии хорошей практикой является создание цепочки полигонов, проходящих по границе формы. Используем данный метод для ретопологии рукава, состоящего из нескольких элементов.
После создания петель ребер для отдельных элементов остается только соединить их с петлей общей формы рукава.

Рис. 1 - Топология рукава

На задней части пальто петли рукава переходят в большее кольцо, описывающее форму лопаток. Обозначение лопаток на спине необходимо для корректной деформации геометрии при движении руки и плеча.
Ребра, составляющие основную часть плаща, соединяются с кольцом рукава с помощью одного треугольника.

Рис. 2 - Топология спины

Учитывая будущую анимацию, скиннинг и развертку создаем топологию каждой из частей моделей, а затем сохраняем их Save Scene As в отдельных файлах формата obj.
