Анимация крокодила на основе ключевых кадров

Auto Key, Set Key

Откроем проект scenes/anim/Croc_Anim_Start.max, который содержит модель крокодила, связанную со скелетом и контроллерами.
Рассмотрим метод анимации в режиме Auto Key Mode. Нажимаем кнопку под таймлайном Auto Key, чтобы начать работу.

Рис. 1 - Панель Timeline

При изменении характеристик объектов на таймлайне будут автоматически созданы ключевые кадры. Типы характеристик, изменения которых будут учитываться, можно настроить в окне Set Key Filters, при нажатии на кнопку Key Filters.
Для создания дополнительных кадров вручную нажимаем клавишу (K).

Рис. 2 - Set Key Filters

Приступим к анимации головы крокодила.
Выделяем объекты контроля головы и челюсти head_ctrl и jaw_ctrl соответственно. Переместимся у нулевому кадру на таймлайне и нажмем (K), чтобы установить изначальное положение. В точке 0 должн

Рис. 3 - Создание ключевого кадра

Теперь переместимся к кадру 20 и повернем два контроллера относительно оси X так, чтобы голова немного поднялась. Так как режим Auto Key включен, на точке таймлайна 20 появится отметка, означающая, что анимация была создана.

Рис. 4 - Создание ключевого кадра

Можем проверить работу анимации, нажав на кнопку воспроизведения Play Animation.

Чтобы анимировать пасть крокодила, выделяем контроллер jaw_ctrl и перемещаемся к кадру 30. Снова поворачиваем объект относительно оси X, чтобы рот открылся. Выключаем режим авктолюча повторным нажатием кнопки Auto Key.

Рис. 5 - Создание ключевого кадра

Далее рассмотрим метод анимации в режиме Set Key. В данном режиме аниматор вручную устанавливает ключевые кадры. Чтобы включить Set Key Mode, необходимо нажать кнопку Set Key.

Рис. 6 - Режим Set Key

Выделяем контроллеры левой передней и правой задней лап l_f_ctrl и r_b_ctrl и перемещаемся к кадру 40. Устанавливаем здесь начальную мозицию анимации, для чего нажимаем клавишу (K) или кнопку Set Keys.

Рис. 7 - Создание ключевого кадра

Перемещаем контроллеры по оси Y и устанавливаем новый ключевой кадр в точке 50.

Рис. 8 - Создание ключевого кадра

Теперь выделяем контроллеры правой передней и левой задней лапы r_f_ctrl и l_b_ctrl. В кадре 50 устанавливаем изначальное положение, а в 60 состояние после перемещения вперед.

В кадре 60 первые две лапы должны возвращаться в изначальное состояние. Выделяем кадр 40 для контроллеров l_f_ctrl и r_b_ctrl и с зажатым Shift перемещаем его в кадр 60. Таким образом произойдет копирование положений. Таким же образом скопируем кадр 50 в 70й для двух других лап.
Чтобы повторить данные движения, выделим все три ключевых кадра и скопируем их так, чтобы дублированный начальный кадр соответствовал конечному. Например, для l_f_ctrl и r_b_ctrl будут созданы следующие кадры: 60 – изначальное положение, 70 – движение вперед, 80 – возвращение в изначальное положение.
