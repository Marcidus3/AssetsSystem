Привязка скелета к модели попугая

Модификатор Skin и Edit Envelopes

Один из необходимых этапов подготовки модели к анимации – скиннинг, т.е. привязка рига к модели. Этот процесс определяет какими частями модели будут управлять элементы рига. Для каждого из его элементов на модели устанавливаются значения веса (степень воздействия).
В 3DS Max скинниг осуществляется посредством модификатора Skin.
Открываем проект scenes/rig/Bird_Skin_Start.max. Здесь имеется скелет с контроллерами и модель, к которой необходимо его привязать.
Для удобства управления скелетом создадим список выделения Selection Set. Выделим все кости и в поле Create Selection Set введем название Bones, нажмем Enter. Кости будут автоматически выбраны при переключении на данный список.

Рис. 1 - Создаем Selection Set

Применяем к модели попугая модификатор Skin, а далее во вкладке Parameters выбираем Bones>>Add.

Рис. 2 - Добавляем модификатор Skin

В открывшемся окне Select Bones выбираем Bones, чтобы автоматически были выделены все кости, а затем нажимаем Select. Теперь кости автоматически привязаны к модели и отображаются во вкладке Parameters.

Рис. 3 - Добавляем кости в модификатор Skin

Можно запустить тестовую анимацию и убедиться, что модель действительно привязана к ригу.
Однако такой автоматический скиннинг недостаточно хорош для использования и создает странные искажения при анимации. Чтобы отредактировать области влияния костей на геометрию модели используется режим Edit Envelopes. При нажатии на кнопку Edit Envelopes и выборе кости из списка Bones для данного модификатора Skin будет выделена область модели, управляемая этой костью. Чем «краснее» цвет области, тем большее влияние производится.

Рис. 4 - Режим Edit Envelopes

Сама кость также окружают две сетки – внешняя и внутренняя. Все точки, находящиеся внутри первой розовой границы, имеют степень воздействия 1. Точки, находящиеся между двумя границами, могут иметь влияние, т.е. вес, от 0 до 1.
Чтобы отредактировать ширину данных границ, достаточно потянуть за управляющие их диаметрами точки. Область Envelope также можно поворачивать и перемещать в пространстве, при использовании концов желтого отрезка, соответствующего кости. При этом сама кость не будет перемещена, только область ее воздействия.

Рис. 5 - Изменение области влияния кости на точки геометрии

Последовательно выделяем элементы скелета и с помощью Edit Envelope настраиваем их эффект на геометрию. Проверяем результат с помощью тестовой анимации.
