Импорт в Unity из 3Ds Max

Чтобы использовать ассет в процессе разработки игры, необходимо импортировать его в игровой движок проекта. Начнем с экспорта модели и ее составляющих из 3Ds Max, где проводилось создание рига и анимации. Выделим все объекты геометрии персонажа и кости. Контроллеры, помощники и IK цепи не должны быть выделены, для удобства скроем слои, на которых они находятся.

Рис. 1 - Выделяем части модели для экспорта

Далее на верхней панели выбираем действие File>>Export>>Export Selected – экспорт выбранных объектов, файл сохраняем в формате .fbx. В окне настроек экспорта обязательно ставим галочки у параметров Animation и Bake Animation во вкладке Animation. Это необходимо для экспорта и запекания анимации.

Рис. 2 - Настройки экспорта

После окончания процесса экспорта можно переходить к движку Unity.

В Unity Hub выбираем опцию создания нового проекта New Project, далее в качестве шаблона выбираем проект 3D из вкладки Core, после чего нажимаем кнопку Create Project.

Рис. 3 - Создание нового проекта

В основном редакторе Untiy нам потребуется создать папки для соответствующих частей ассета. Нажимаем правой кнопкой мыши на пустом пространстве панели Project. В нижней части экрана, выбираем Create>>Folder. Задаем имя новой папке и нажимаем клавишу Enter. Иерархия созданных папок будет отображаться в левой части панели. Корневой папкой будет Assets.

Рис. 4 - Создание папки

Пусть папка Models содержит все ассеты проект, Materials материалы и текстуры, а Animation анимации и их контроллеры. Чтобы импортировать модель и текстуры, перетаскиваем их из папки экспорта в соответствующие папки в Unity. Файл модели будет иметь превью, при нажатии на которое будет отображены ее составляющие: отдельные элементы геометрии, анимация, риг.

Рис. 5 - Импортированные объекты

Также при нажатии на файл правая панель Inspector отобразит информацию о ней и ее составляющих. Во вкладке Animation меняем название импортированной анимации на Idle, так как это анимация покоя, и ставим галочку у Loop Time для включения зацикливания. Нажимаем Apply для применения изменений.

Рис. 6 - Настройка импортированной модели

Теперь осталось добавить ассет в игру. Для этого перетаскиваем импорт из панели Project в рабочее пространство движка. С помощью стандартного элемента управления Gizmo или панели Inspector перемещаем и поворачиваем модель.
