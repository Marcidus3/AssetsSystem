Контроль пальцев

Для управления пальцами будем использовать атрибуты, определяющие порот их фаланг.
Создадим для контроллера IK еще несколько атрибутов в модификаторе Attribute Holder. Каждый из них будет управлять сгибанием и разгибанием соответствующего пальца.

Рис. 1 - Атрибуты

Чтобы связать значения параметров с поворотом костей используем Wire Parameter. Как основной параметр в левой части окна Parameter Wiring будет зависеть от пальца который на данный момент. Т.е. параметр Index_f будет вызван только в случае, если происходит настройка указательного пальца.
Зависимыми от атрибута должны быть параметры поворота каждой из трех фаланг пальца. Формула, определяющая зависимость атрибута от угла поворота, задается для каждого пальца и фаланги отдельно, так, чтобы палец сгибался реалистично.
