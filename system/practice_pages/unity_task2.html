Материалы Shader Graph

Теперь, когда геометрия импортирована, необходимо применить к ней материалы.
Перетащим текстуры ассета в папку Material. В данной папке нажимаем правой кнопкой мыши в выбираем создание нового материала Create>>Material, задаем его имя.

Рис. 1 - Создание нового материала

Так как стандартные материалы Unity не всегда идеально подходят под нужды разработчиков, используется дополнительный режим настройки отображения Shader Graph. Чтобы установить данный функционал, откроем окно загрузки дополнительных пакетов Window>>Package Manager и в разделе Packages:Unity Registry выберем Shader Graph, нажмем Install для установки.

Рис. 2 - Добавление пакета Shader Graph в проект

Теперь при создании нового объекта в одной из папок будет доступен перечень новых режимов отображения материалов и их настройки. В папке Material создаем объект Shader Graph, Builtin, Unlit Shader Graph.

Рис. 3 - Создаем новый шейдер

Перетащим иконку данного Shader Graph на иконку материала, теперь на панели Inspector данного материала в качестве шейдера Shader будет указан только созданный нами.

Рис. 4 - Добавляем шейдер к материалу

Кликаем два раза по иконке Main_Shader, чтобы открыть окно настройки. Здесь узел Fragment - основной материал. Он содержит атрибуты (на данный момент доступен только цвет Base Color) и может быть настроен в окне Graph Inspector справа.

Рис. 5 - Окно Shader Graph

Чтобы добавить параметры выделим основной узел и нажмем клавишу Пробел. Откроется окно выбора параметров узла Create Node, кликнем дважды по одному из вариантов, и он добавиться в Fragment. Таким образом добавим следующие параметры: Normal – нормали, Emission – свечение и Metallic металлизированные области.

Рис. 6 - Добавление параметров

Перетащим текстуры с панели Project в окно Shader Graph. От точки выхода RGBA новых узлов текстур проведем соединительную линию к соответствующим параметрам узла Fragment. Теперь текстуры присоединены к главному материалу.

Рис. 7 - Текстуры

Нам требуется уменьшить силу карты нормалей для данной модели. Чтобы осуществить это, нажмем правой кнопкой в рабочей области Graph Editor, выберем функцию Create Node и в открывшемся окне со список доступных к созданию узлов выберем узел Normal Strengh – сила нормалей. Узел появится в рабочей области.

Рис. 8 - Добавление узла Normal Strengh

Соединим узел текстуры нормалей с точкой In узла Normal Strengh, чтобы свзять их. Измненим параметр Strengh данного узла и увидим, что превью текстуры изменяется. Теперь соединим точку выхода Out с точкой Normal узла Fragment. Теперь нормали, воздействующие на модель, изменят свою силу, при этом изначальный файл текстуры не исказится.

Рис. 9 - Соединяем узлы Normal Map и Normal Strengh

Чтобы применить материал к модели, вернемся к рабочей области проекта Scene и перетащим иконку материала на все части модели (в данном случае на основную геометрию тела и глаза).
