Создание скелета персонажа

Импортируем модель в 3DS Max помощью функции File>>Import>>Import.
Перейдем к модификатору Editable Mesh модели головы, включим режим Attach – присоединить, и кликнем на все остальные объекты кроме глаз.
Прежде чем начать создавать кости, сделаем модель прозрачной, выделим ее и нажмем комбинацию клавиш Alt+X, чтобы включить режим X-Ray. Кликнем по модели правой кнопкой мыши и выберем Freeze Selection, так модель будет недоступна для изменения и не помешает при создании костей.

Рис. 1 - Режим X-Ray

Скелет, который будет создаваться для данного ассета, практически полностью соответствует ригу из задания «Создание простого рига (Bone Tool)». Такой риг подойдет для большинства двуногих существ и потребует минимальные изменения, чтобы подогнать его к модели.
Изменение положения костей происходит строго в режиме Bone Edit Mode, после завершения работы с конечной костью, ее окончание End удаляется и создается заново с помощью кнопки Create End. Это позволяет сохранять ориентацию всех костей корректной. Также сбрасывается растяжение костей с помощью кнопки Reset Stretch для выделенных костей.
Здесь основной костью корнем всего скелета является таз pelvis, к которому присоединена цепь позвоночника spine base, spine 1,2, кость шеи neck и головы head.
К тазу присоединены две кости бедер thigh, которые являются корнями для цепей ног. Ноги состоят из костей бедер, лодыжек Calf, кости основной части ступни Heel и кости пальцев ног toe.
Кости ключиц clavicle присоединяются к кости позвоночника spine 2, идущей перед шеей. От костей ключиц соответственно отходят кости плеча shoulder и предплечья forearm, кончик цепи определяет поворот запястья и необходим для дальнейшего риггинга.
Также важно сохранять углы между определенными костями, чтобы инверсивная кинематика правильно работала, например колени и локти должны быть согнуты.

Рис. 2 - Иерархия костей

Так как кости рук требует наличия прямой и обратной кинематики для реалистичной анимации, нам необходимо создать соответствующие им цепи.
Выделим цепь руки без ключицы двойным кликом по кости плеча. Используем функцию Edit>>Clone, чтобы скопировать ее в режиме Copy. Добавим к названию новых костей префикс FK, так как они будут относиться к цепи прямой кинематики.
Увеличим кости в окне Bone Tools и поменяем их цвет.

Рис. 3 - Клонируем кости

Таким же образом создадим копии оригинальных костей для обратной кинематики, префикс изменим на IK, а сами кости уменьшим и также изменим цвет.
Так как при изменении размера костей их окончания деформируются, удалим их и создадим снова.

Рис. 4 - Кости запястья кажой из цепей

Скопируем цепи на правую часть модели с помощью инструмента Mirror. Кости для цепей IK и FK готовы. Переходим к созданию рига кистей рук.
Создаем основную кость для ладони и пальцы на ее передней поверхности в виде Front, чтобы сохранять для них правильную ориентацию. Первый созданный палец копируем три раза и корректируем, чтобы сэкономить время.
Большой палец создается в виде Top также для сохранения ориентации.

Рис. 5 - Создание костей кисти руки

Первые фаланги всех пальцев выделяются, а затем с помощью инструмента Select and Link присоединяются к ладони. Ладонь в свою очередь присоединяется к кости запястья.

Рис. 6 - Соединение костей пальцев и ладони

Для подола плаща был создан набор из трех костей для левой, правой и центральной его частей. Данные кости привязаны к позвоночнику spine2.
