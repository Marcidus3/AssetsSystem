Анимация хвоста

Key Properties

Продолжаем работать с крокодилом из прошлого задания или открываем проект scenes/anim/Croc_Anim2_Start.max.
В данном задании необходимо будет анимировать хвост крокодила, однако среди 100 доступных кадров на таймлайне заняты 90. Чтобы увеличить количество кадров и длительность анимации, откроем окно настройки времени Time Configuration.

Рис. 1 - Открываем окно Time Configuration

В данном окне нам потребуется раздел Animation, в котором содержаться значения количества кадров, начала и длины анимации. Здесь в поле End Time установим значение 220, параметр Frame Count, определяющий кадры, автоматически примет значение 221.

Рис. 2 - Time Configuration

Для управления хвостом потребуется возимодействовать с костями внутри модели. Чтобы сделать ее прозрачной, можно переключить отображение в данном виде в режим Wireframe Override, либо включить режим Xray, нажав сочетание клавиш Alt+X.

Рис. 3 - Включаем режим отображения Wireframe

Рис. 4 - Режим X-Ray

Чтобы не допустить нечаянного выделения модели в процессе анимации костей, необходимо ее «заморозить». Выделаем ее и нажимаем правой кнопкой мыши, затем выбираем Freeze Selection. Теперь можно спокойно выделять кости.

Рис. 5 - Freeze Selection

Для дальнейшейго создания анимации можно использовать любой из режимов (Auto Key или Set Key).
В кадре 90 устанавливаем изначальное состояние костей tail01, tail02 и tail03. Повороачиваем каждую из костей относительно оси Z вправо и сохраняем данное положение в кадре 120.

Рис. 6 - Поворот хвоста

Следующей ключевой позицией будет поворот хвоста в обратном направлении. Сначала скопируем кадр 120 в 160й, а затем нажмем на него правой кнопкой мыши, чтобы открыть перечень действий над ключом. Категория Key Properties содержит все параметры выделенных объектов, которые изменются/могут быть изменены в рамках анимации.
Так как выделены все три кости, мы видим параметры положения, поворота и размера для каждой из них.

Рис. 7 - Выбираем Key Property кадра

Нажмем на вариант tail03: Z Rotation, т.е. поворот большой кости хвоста относительно оси Z. В открывшемся окне можно настроить значение характеристики Value и кадр Time, которому она будет соответствовать, а также ее кривые.
В нашем случае значение Value показывает угол, на который была повернута кость. Добавляем минус перед углом, чтобы поворот отразился относительно главой оси модели.

Рис. 8 - Устанавливаем значение параметра

Ту же операцию проделаем с параметрами tail02: Z Rotation, и tail01: Z Rotation.

Рис. 9 - Повернули хвост с помощью Key Properties

Скопируем изначальные положения хвоста и головы в последний кадр 220, чтобы зациклить анимацию.
Нажмем правой кнопкой мыши в любой области экрана и выберем Unfreeze All, чтобы сделать модель крокодила доступной для взаимодействия.
