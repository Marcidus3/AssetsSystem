Создание зависимостей и анимация шестеренок

Wire Parameters

Открываем проект scenes/anim/Mouse_Spin_Start.max В рамках данного задания нам необходимо установить зависимость между вращением большой и малой шестеренок, а также движением все модели.
В данном проекте большим колесом управляет объект оси pin01, выделяем его и нажимаем правой кнопкой мыши, чтобы затем в списке выбрать Wire Parameters.
Wire Parameters – функция, позволяющая связать два объекта так, чтобы параметры одного изменяли другого автоматически.
Для pin01 выбираем параметр зависимости X Rotation – поворот относительно оси X (Wire Parameters>>Transform>>Rotation>>Zero Euler XYZ>>X Rotation).

Рис. 1 - Выбираем параметр Wire Parameters для объекта

Тянем пунктирную линию к оси малой шестеренки pin00, для нее устанавливаем такой же параметр зависимости X Rotation.

Рис. 2 - Соединяем параметры объектов

В открывшемся окне Parameter Wiring отображаются объекты и их параметры, их связь и уравнения, определяющие ее. В левом окне выбран объект pin01, в правом pin00. Чтобы установить зависимость между ними, нажимаем кнопку со стрелочкой слева направо (control direction), а затем кнопку Connect. Управляющая характеристика подсветится зеленым, управляемая – красным.
В правом нижнем окне отображается формула, которая будет определять зависимость параметров. По умолчанию зависимость прямая, а значит формула = X_Rotation.

Рис. 3 - Настройка Wire Parameters

Проверим анимацию и заметим, что, во-первых, вращение обеих шестеренок происходит в одном направлении, а во-вторых, их скорости не соответствуют друг другу.

Рис. 4 - Результат

Чтобы вращение происходило в обратные стороны добавим минус перед параметром X_Rotation.
Так как большая шестеренка имеет в два раза больше зубьев, она будет вращаться в два раза медленнее. Умножаем параметр поворота X_Rotation на 2.
Осталось установить шестеренки в такое положение относительно друг друга, чтобы создать иллюзия нажатия зубьев бОльшей на малую. Для этого подбираем число, которое необходимо прибавить к основному параметру. В данном случае подошло число -0.61.

Рис. 6 - Результат

Теперь необходимо установить зависимость поворота колеса от перемещения модели.
Окно настроек Wire Parameter также можно открыть через вкладку Animation>>Wire Parameters>>Parameter Wire Dialog.

Рис. 7 - Открываем окно Wire Parameters

В окнах выбора отображаются все возможные параметры, которые можно связать между собой. Находим части модели во вкладке Objects. Первый объект – body, так как к нему привязаны все остальные части мыши, параметр – Y Position. Второй – pin01 с параметром X Rotation.
Здесь перед Y_Position ставим минус, чтобы поменять направление вращения, и умножаем на 0.15, чтобы уменьшить поворот.
